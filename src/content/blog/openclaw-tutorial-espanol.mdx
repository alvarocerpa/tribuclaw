Cuando empecé con OpenClaw, el mayor obstáculo no fue la tecnología — fue la documentación. Toda en inglés, dispersa en varios repositorios, sin un camino claro de principio a fin.

Este tutorial es lo que me habría gustado encontrar cuando empecé. En español, ordenado, con cada paso explicado para que no te pierdas en ningún momento.

## Lo que necesitas antes de empezar

Antes de instalar nada, asegúrate de tener:

**Un VPS con Ubuntu 22.04 o superior.** Recomiendo Hetzner (CX22, 3,79€/mes) para empezar. Es el equilibrio perfecto entre precio y rendimiento. DigitalOcean y Contabo también funcionan bien.

**Una API key de un modelo de IA.** Anthropic (Claude) es la mejor opción. Regístrate en console.anthropic.com, añade créditos (5 dólares son suficientes para empezar) y genera tu clave.

**Una cuenta de Telegram.** Es el canal principal para comunicarte con tu agente. Necesitas crear un bot con @BotFather y guardar el token.

**Node.js 20 o superior** en el servidor. Si usas Ubuntu, se instala en dos comandos.

## Instalación paso a paso

### Paso 1: Conectar al servidor

```bash
ssh root@tu-ip-del-servidor
```

### Paso 2: Instalar Node.js

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### Paso 3: Instalar OpenClaw

```bash
npm install -g openclaw
```

### Paso 4: Configuración inicial

```bash
openclaw configure
```

Este comando interactivo te pregunta tu API key del modelo de IA, el token del bot de Telegram y otras opciones básicas. Responde las preguntas y OpenClaw se configura solo.

### Paso 5: Arrancar el gateway

```bash
openclaw gateway start
```

El gateway es el proceso que mantiene la conexión entre tu agente y tus canales de comunicación. En producción lo conviertes en un servicio del sistema para que arranque automáticamente.

## Personalizar tu agente

La instalación base es funcional, pero el valor real viene de personalizar el agente para que te conozca a ti.

**SOUL.md** — La personalidad de tu agente. Aquí defines cómo habla, qué valores tiene, cómo responde cuando comete errores. Es el documento más importante para que tu agente suene natural.

**USER.md** — Información sobre ti. Nombre, negocio, preferencias, contexto. Cuanto más sepa de ti, mejor te servirá.

**MEMORY.md** — El cerebro permanente. Aquí va el conocimiento destilado que quieres que recuerde siempre: decisiones tomadas, preferencias importantes, proyectos activos.

**HEARTBEAT.md** — Qué hace el agente cuando tú no le estás hablando. Checks automáticos, tareas recurrentes, monitorización.

## Conectar Telegram

Es el paso más directo. Con el token del bot de BotFather ya configurado en el paso anterior, solo tienes que:

1. Buscar tu bot en Telegram
2. Mandarle `/start`
3. El agente responde y estáis conectados

A partir de ese momento, todo lo que mandes al bot llega a tu agente. Y todo lo que el agente quiera comunicarte llega a tu Telegram.

## Los primeros días

Las primeras 24-48 horas son de aprendizaje mutuo. El agente procesa las instrucciones que le has dado y empieza a construir su modelo de cómo trabajas. Cuéntale cosas de tu negocio, cuéntale qué tareas quieres que gestione, prueba distintas formas de pedirle cosas.

Cuanto más lo uses al principio, más rápido aprende y mejor funciona.
