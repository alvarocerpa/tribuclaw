## Cómo instalar OpenClaw: guía paso a paso

Aquí va lo prometido: **instalar OpenClaw** en menos de 15 minutos. La instalación real es via terminal (un comando), pero te lo explico tan simple que cualquier persona lo sigue, aunque nunca haya usado la terminal.

Si algún día quieres una instalación 100% técnica en servidor VPS para que funcione 24/7, tenemos una [guía avanzada paso a paso](/blog/instalar-openclaw-en-vps-paso-a-paso). Pero para empezar, lo haremos en tu propia PC.

### Paso 1: Prepara tu equipo (2 minutos)

Necesitas:
- PC con Windows 10+, Mac con macOS 11+ o Linux básico (como Ubuntu).
- 4GB RAM mínimo (más es mejor).
- Conexión internet.

Nada raro, ¿verdad?

### Paso 2: Abre la terminal (1 minuto)

Sí, suena técnico, pero es solo abrir un programa:

- **Windows**: Busca "Símbolo del sistema" o "PowerShell" en el menú inicio.
- **Mac**: Busca "Terminal" en Spotlight (Cmd + Espacio).
- **Linux**: Ctrl + Alt + T.

Verás una pantalla negra con texto. No te asustes, es normal.

### Paso 3: Instala Node.js (5 minutos)

OpenClaw necesita Node.js para funcionar. Es como el motor.

1. Ve a [nodejs.org](https://nodejs.org)
2. Descarga la versión "LTS" (recomendada)
3. Instala como cualquier programa: Siguiente, Siguiente, Finalizar.

Cierra y vuelve a abrir la terminal para que reconozca Node.

Verifica escribiendo: `node --version` (debe mostrar un número como v20.x.x)

### Paso 4: Instala OpenClaw (3 minutos)

En la terminal, copia y pega este comando:

```bash
npm install -g openclaw
```

Pulsa Enter. Verás que descarga cosas. Espera a que termine (un minuto aprox).

### Paso 5: Inicia OpenClaw (2 minutos)

Escribe en la terminal:

```bash
openclaw setup
```

Se abrirá un asistente que te guiará:
- Crear tu agente (ponle nombre, ej: "Mi asistente")
- Conectar Telegram o WhatsApp (escanea QR como WhatsApp Web)
- Configurar API de IA (OpenAI, Anthropic, etc.)

¡Listo! Tu **asistente IA personal para negocios español** está vivo.

## Solución de problemas comunes durante la instalación y uso

Aunque la instalación es sencilla, a veces surgen tropiezos. Aquí te detallo los errores más habituales y cómo solucionarlos rápido, sin estrés.

### Errores comunes en la instalación

- **"node no encontrado" o "npm no reconocido"**: Node.js no está en el PATH. Solución: Reinicia la terminal, o reinstala Node marcando "Add to PATH". En Windows, usa PowerShell como admin.
- **Permisos denegados en Mac/Linux**: Añade `sudo` antes del comando: `sudo npm install -g openclaw`. Te pedirá contraseña.
- **Descarga lenta o falla**: Usa VPN o cambia DNS a 8.8.8.8 (Google). O prueba `npm cache clean --force` primero.

### Problemas de conexión con WhatsApp

- **QR no carga o expira**: Refresca la página (F5), asegúrate de que WhatsApp Web no esté abierto en otro sitio. Limpia cache del navegador si persiste.
- **Sesión desconectada**: WhatsApp detecta bots; usa el modo "headless" en config o reconecta manual. Consejo: No abuses de envíos masivos al inicio.
- **Mensajes no llegan**: Verifica que el número esté autorizado en WhatsApp Business API (si usas eso) o que el puerto 3000 no esté bloqueado por firewall.

### ¿Qué hacer si la API de IA falla?

- **"Invalid API key"**: Copia-pega de nuevo la clave desde OpenAI dashboard. Chequea si es GPT-4o o compatible.
- **Sin créditos o rate limit**: Añade saldo o espera 1min. Alternativa: Cambia a Grok (x.ai) gratis para pruebas.
- **Respuestas lentas o errores 500**: Baja el modelo a "gpt-3.5-turbo" temporalmente, o chequea tu conexión (prueba `ping api.openai.com`).
